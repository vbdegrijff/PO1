HTML en CSS

HTML en CSS zijn de bouwstenen van websites. Met HTML (Hypertext Markup Language) bepaal je wat er op een pagina komt, zoals tekst en afbeeldingen, en met CSS (Cascading Style Sheets) geef je alles een mooi uiterlijk, zoals kleuren en lettertypes. Samen zorgen ze ervoor dat een website werkt en er goed uitziet!

Wat is HTML?
HTML gebruikt 'tags' om dingen op een pagina te maken, zoals:
1. <h1> t/m <h6>: Voor koppen.
2. <p>: Voor teksten of paragrafen.
3. <a>: Voor links (naar andere websites).
4. <img>: Voor afbeeldingen.
5. <div>: Voor het indelen van de pagina.
6. <li, ul en ol> Voor lijsten (genummerd en niet genummerd)
7. <table>: Voor tabellen.
8. <br>: Om de tekst over te zetten op de volgende regel.
9. <I, U, B>: Voor Italic, Underline, en Bold.

Met CSS kun je deze dingen mooi maken. Je kunt bijvoorbeeld kiezen welke kleuren of lettertypes je wilt gebruiken, en je kunt de grootte of positie van dingen aanpassen. Zo zorg je ervoor dat je site er precies zo uitziet als jij wilt.
